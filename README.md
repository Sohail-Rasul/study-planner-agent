# Smart Study Planner Agent

AI-powered study planning assistant built using Google Agent Development Kit.

## Features

- Study planner
- Pomodoro scheduling
- Memory
- MCP integration
- Agent Skills
- Streamlit UI

## Tech

- Python
- FastAPI
- Streamlit
- Gemini
- ADK
- MCP

## Running

pip install -r requirements.txt

streamlit run frontend/app.py
